Bienvenue dans le projet universitaire B !

Ce projet a pour but la réalisation d'une application web pour la gestion des stages en entreprise. Trois types de profils peuvent accéder à cette application:
- Les étudiants de l'Université de Bordeaux à la recherche d'un stage
- Les Entreprises proposant des offres pour recruter des futurs candidats.
- Les administrateurs habilités à gérer les accés et à valider les offres.

Liens

- Codes sources sur GitHub: https://github.com/emmapriyanka/ProjetUnivB
- Site déployé avec Render (Veuillez noter que le site peut être occasionnellement en maintenance sur Render): https://projetunivb.onrender.com/accueil


Dossier Documentation

Toute la documentation relative au projet se trouve dans le dossier "Documentation". Veuillez consulter les fichiers disponibles dans ce dossier
pour obtenir des informations détaillées sur le fonctionnement du projet (cahier des charges, manuel utilisateur etc).

Base de donnés (Dossier sql)

Toute la documentation correspondant à la création et l'initialisation de la base se trouve dans le dossier sql ainsi qu'un petit jeu de données.
Le jeu de données est un ensemble de fichiers csv pour les utilisateurs, entreprises et offres de stage qui seront chargés en base avec l'outil psql.
Ceci permettra d'avoir un jeu de données pour tester l'application.
